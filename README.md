# ex-plugin

- 뷰 플러그인 프로퍼티 중복시 처리되는 우선 순위 예제

> Vue 스터디 과제

### 테스트하기

``` bash
# clone source
git clone https://github.com/denlyou/example-vue-plugin-priority.git
# move directory
cd example-vue-plugin-priority
# install dependencies
npm install
# serve with hot reload at localhost:8080
npm run dev
```
